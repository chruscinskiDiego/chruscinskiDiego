## 👨‍💻 About Me

I'm a **Mid-Level Software Developer at Linx**, with a degree in **Systems Analysis and Development (UTFPR)**.

I work as a **Full Stack Developer**, participating in the entire application lifecycle, from interface development to data infrastructure, with a primary focus on **back-end engineering**.

On a daily basis, I build **scalable, secure, and high-performance APIs**, applying software architecture best practices, clean code organization, and long-term maintainability principles.

I have solid experience in the **Node.js** ecosystem, with strong expertise in **NestJS** and **TypeScript**, as well as consistent experience with relational databases such as **SQL Server** and **PostgreSQL**, including data modeling, query optimization, and ORM usage with **TypeORM**.

My full stack experience with **React** gives me an important systems-level perspective, helping me design APIs that are clearer, more efficient, and easier to integrate with the front end.

I also have hands-on experience with **Infrastructure and DevOps**, including:
- **Docker**
- **CI/CD with Azure DevOps**
- **Cloud servers (OCI and Azure)**
- **HashiCorp Vault**
- **MinIO**
- **Amazon S3**
